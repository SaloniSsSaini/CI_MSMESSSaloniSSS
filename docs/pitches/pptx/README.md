## PowerPoint pitch decks

This folder contains generated PowerPoint decks created from the pitch narratives and UI mock screenshots in `docs/pitches/assets/`.

### Generate / refresh the decks

Run:

```bash
python3 /workspace/docs/pitches/build_pitches_pptx.py
```

Outputs:
- `Carbon_Intelligence_Investor_Pitch.pptx`
- `Carbon_Intelligence_MSME_Pitch.pptx`
- `Carbon_Intelligence_Bank_Pitch.pptx`
- `Carbon_Intelligence_Key_Employees_Partners_Channels_Pitch.pptx`
